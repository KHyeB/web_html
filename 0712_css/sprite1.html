sprite images

sprite images는 규모가 있는 중견기업 및 대형쇼핑몰등에서 많은 아이콘 이미지 (100px이하) 를 효율적으로 관리하기 위해서 제작하는 이미지이다.

img 태그가 아닌 CSS에서 background-image 로 연결해서 사용한다

날씨:비 검색
메뉴 장바구니 관심상품 배송지등록 검색 kurly
  1. 1. 아이콘의 의미를 HTML태그에 작성한다. ex) 검색, 바로가기, 홈, 위로, 아래로
  2. 2. 위 HTML을 선택자로 만들어서 background-image로 sprite이미지를 연결한다
  3. 3. sprite image 개별위치, 크기를 체크해서 css로 입력한다.
  4. 4. (1)에서 작성한 의미있는 글자는 text-indent로 숨긴다.
